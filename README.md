# 🧩 LeetCode Solutions Repository

This repository contains my accepted solutions to LeetCode problems as part of my Data Structures & Algorithms and interview preparation journey.

## 🚀 About

The goal of this repository is to consistently practice coding problems, improve problem-solving skills, and track my progress on LeetCode.

Each solution is synced directly from my LeetCode account and serves as a record of my learning and practice.

## 🛠️ Language Used

* C++ (Primary)
* More languages may be added in the future

## 📂 Repository Structure

Each problem is stored in its own folder and typically contains:

* Solution source code
* Problem statement (`README.md`)

The folders are named using the LeetCode problem number followed by the problem title.

### Example

```text
0001-two-sum/
├── README.md
└── Solution.cpp

0009-palindrome-number/
├── README.md
└── Solution.cpp
```
